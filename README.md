# QR Code Generator 🌌

A web-based QR Code Generator built with Python and Flask.
Enter any URL, choose a filename, and download your QR code instantly as a PNG file.

## Tech Stack
- Python
- Flask
- qrcode library
- HTML & CSS (Galaxy Theme)

## Features
- Generate QR codes from any URL
- Instant PNG download
- Clean galaxy-themed UI
